# GrassRootsBBS 説明書

## 本家 Big-model との違い

### サーバ設定(config.toml)

昔の BBS のように少数回線にする場合、NORMAL_SSH_PORT_COUNT を回線数、MAX_CONCURRENT_NORMAL_SSH_CLIENTS を 1 にしてください。

#### 3 回線ホストの再現

NORMAL_SSH_PORT_COUNT = 3 # 通常接続のポート数  
MAX_CONCURRENT_NORMAL_SSH_CLIENTS = 1 # 通常接続の同時接続可能クライアント数(0 で無制限)

ただし、ローカルログインができないのでシスオペも 1 回線埋めてしまうことに注意してください。
