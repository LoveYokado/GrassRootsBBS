総合動作確認チェックリスト
1. 基本動作とログイン
[ ] 起動: アプリケーションがエラーなく正常に起動するか。
[ ] ログイン:
    [ ] 通常ユーザーでログインできるか。
    [ ] シスオペ (Sysop) でログインできるか。
    [ ] GUESTでログインできるか。
[ ] マルチログイン防止: 同じユーザーID（GUEST以外）で、別のブラウザやタブからログインしようとすると、後から接続した方が拒否されるか。
[ ] ログアウト: 正常にログアウトし、ログイン画面に戻るか。
2. Webターミナルの表示と操作
[ ] にじみ効果:
    [ ] 設定ポップアップ (F1) で「にじみ (Glow)」効果をON/OFFできるか。
    [ ] ONにした際、文字に赤と緑の「色にじみ」が適用されているか。
    [ ] 各テーマ（Default, Green, Amber）で、それぞれに適したにじみ効果になっているか。
[ ] バックスペース:
    [ ] 日本語などの全角文字を入力中にバックスペースキーを押すと、1文字分が正しく削除されるか。
[ ] ファンクションキー:
    [ ] F1: 設定ポップアップが開くか。
    [ ] F2: ログ記録が開始/停止できるか。（LOGGING... の表示に切り替わるか）
    [ ] F3: ログビューワーが開くか。
    [ ] F5: 1行エディタが開くか。
    [ ] F6: マルチラインエディタが開くか。
3. 掲示板 (BBS)
[ ] メニュー: bキーで掲示板の階層メニューに入れるか。
[ ] 記事一覧:
    [ ] 掲示板を選択して、記事一覧が表示されるか。
    [ ] j/kキーや矢印キーでカーソルが移動できるか。
[ ] 記事の閲覧:
    [ ] Enterキーで記事が読めるか。
    [ ] スレッド形式の掲示板で、返信が正しく表示されるか。
[ ] 記事の投稿:
    [ ] wキーで新規記事（スレッド）が投稿できるか。
    [ ] 記事閲覧中にrキーで返信が投稿できるか。
[ ] 記事の削除/復元:
    [ ] *キーで自分の投稿を削除できるか。
    [ ] 削除された記事に再度*キーで復元できるか。
    [ ] シスオペは他人の投稿も削除/復元できるか。
4. チャット
[ ] メニュー: cキーでチャットルーム選択メニューに入れるか。
[ ] 入退室:
    [ ] チャットルームに入室できるか。
    [ ] ^キーで退室できるか。
    [ ] 入退室メッセージが他の参加者にブロードキャストされるか。
[ ] メッセージ送受信:
    [ ] メッセージが自分と他の参加者の画面に表示されるか。
[ ] コマンド:
    [ ] !wでオンラインメンバーが表示されるか。
    [ ] !lでルームをロックできるか。
    [ ] !uでロックを解除できるか。
5. メール
[ ] メニュー: mキーでメールメニューに入れるか。
[ ] 受信:
    [ ] Rキーで新着メールを1件ずつ読めるか。
    [ ] （重要） Rキーで読んだ際、送信者名が「(不明)」にならず、正しく表示されるか。
[ ] 一覧:
    [ ] Lキーでメール一覧が表示されるか。
    [ ] 一覧からメールを選択して読めるか。
[ ] 送信:
    [ ] Wキーでメールを作成し、送信できるか。
6. シスオペメニュー
[ ] メニュー: sキーでシスオペメニューに入れるか。
[ ] ユーザー管理:
    [ ] usrl: ユーザー一覧が表示されるか。
    [ ] regs: 新規ユーザーが登録できるか。
    [ ] delu: ユーザーを削除できるか。
[ ] 掲示板管理:
    [ ] lsbd: 掲示板一覧が表示されるか。
    [ ] mkbd: 新規掲示板が作成できるか。
    [ ] dlbd: 掲示板を削除できるか。
