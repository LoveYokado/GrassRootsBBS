GrassRootsBBS 運用マニュアル

このドキュメントは、GrassRootsBBSの利用方法と管理方法について説明します。
サーバーのインストールとセットアップについては、README.md を参照してください。

1. For Users / ユーザー向けマニュアル

このセクションでは、BBSの基本的な使い方を説明します。

1.1. BBSへのアクセス

WebブラウザからBBSのURL（例: http://<サーバーのIPアドレス>:5000）にアクセスします。
ログイン画面が表示されたら、ユーザーIDとパスワード、またはPasskeyを使用してログインしてください。

1.2. 主な機能

掲示板 (BBS)

  • 読み書き: r で記事一覧を読み、w で新しい記事を投稿します。
  • 探索リスト: 記事一覧画面で @ を押すと、その掲示板を「探索リスト」に追加/削除できます。トップメニューの n (新アーティクル探索) などで、リストに登録した掲示板だけを効率的に巡回できます。
  • ファイル添付: ファイルアップロードが許されている掲示板では、記事投稿時、本文入力後にファイルを添付できます。

チャット (Chat)

  • 入室: メニューからチャットルームを選択して入室します。
  • コマンド: チャット中に !? を入力すると、利用可能なコマンド（退室、オンラインメンバー表示など）のヘルプが表示されます。

Passkey (パスワードレス認証)

Webターミナルでログイン後、ユーザー設定 (u コマンド) の「Passkey 管理」から、顔認証や指紋認証、USBセキュリティキーなどを使ってログインするためのPasskeyを登録できます。登録後は、パスワードなしで安全にログインできるようになります。

2. For System Operators / 管理者向けマニュアル

このセクションでは、BBSの管理と運営に関するタスクを説明します。

2.1. 管理画面へのアクセス

シスオペは、以下のURLからWebベースの管理画面にアクセスできます。

  • http://<サーバーのIPアドレス>:5000/admin

ログイン後、ユーザー管理、掲示板管理、システム設定など、BBSのあらゆる側面を直感的に管理できます。

2.2. 主な管理タスク

BBSのほとんどの設定は、Web管理画面から直感的に行うことができます。

  • 掲示板とチャットルームの構成: setting/bbs_mode3.yaml や setting/chatroom.yaml を編集後、管理画面の「Board Management」から掲示板の作成・編集を行ってください。
  • ログインメッセージの変更: 管理画面の「System Settings」から変更できます。
  • バックアップ: 管理画面の「Data Management」から手動・自動バックアップの設定が可能です。
  • Push通知: Push通知機能を使用するには、VAPIDキーペアの生成と設定が必要です。詳細は README.md を参照してください。
